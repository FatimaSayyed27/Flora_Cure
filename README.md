# 🌱 FloraCure – AI-Powered Plant Doctor

FloraCure is an *expert system built with Django + TailwindCSS* that helps diagnose plant diseases, track history, and manage user profiles with a clean, modern interface.

---

## ✨ Features
- 🏠 *Home Page* – Elegant emerald theme with glassmorphism.
- 📊 *Dashboard* – Weather integration, stats cards, recent diagnoses.
- 🔍 *Diagnose* – Upload plant images, select symptoms, get AI-based diagnosis.
- 📜 *History* – Track past diagnoses.
- 👤 *Profile* – Update name, email, password, and city.
- 🔒 *Authentication* – Secure login/logout with Django’s built-in auth.

---

## 🛠️ Tech Stack
- *Backend:* Django (Python 3.12)
- *Frontend:* TailwindCSS
- *Database:* SQLite (default, can switch to PostgreSQL/MySQL)
- *Auth:* Django User model + custom Profile model
